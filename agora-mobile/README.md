# Agora Mobile App

Parent & Student mobile app for the Agora School Management Platform.

## Tech Stack

- **Framework:** Flutter 3.16+
- **Language:** Dart
- **State Management:** Provider
- **HTTP:** http package
- **Charts:** fl_chart

## Setup

```bash
cd agora-mobile
flutter pub get
flutter run
```

## API Configuration

Edit `lib/core/config.dart`:

```dart
// Android emulator -> localhost
static const String apiBaseUrl = 'http://10.0.2.2:8080/api/v1';

// iOS simulator
static const String apiBaseUrl = 'http://localhost:8080/api/v1';

// Physical device (use your machine's LAN IP)
static const String apiBaseUrl = 'http://192.168.x.x:8080/api/v1';
```

## Screens

| Screen | Description | Tab |
|--------|-------------|-----|
| Login | School code + email + password with demo credentials | - |
| Dashboard | Stats cards, user info, quick overview | Home |
| Attendance | Date-filtered attendance records with stats | Attendance |
| Homework | Assignment list with due dates, overdue markers | Homework |
| Marks | Assessment list with type badges and max marks | Marks |
| Messaging | Conversation list + chat thread (WhatsApp-style) | Messages |
| Notifications | Push notification history with read/unread | AppBar icon |

## Project Structure

```
lib/
  main.dart              # App entry point
  core/
    config.dart           # API URL configuration
    theme.dart            # Colors, typography, component themes
    api_client.dart       # HTTP client with JWT auth
    widgets.dart          # Shared UI components
  models/
    user.dart             # User model
    attendance.dart       # Attendance record model
    homework.dart         # Homework model
    assessment.dart       # Assessment + score models
    conversation.dart     # Conversation + message models
    notification_item.dart # Notification model
  providers/
    auth_provider.dart    # Login, logout, auto-login state
  screens/
    login_screen.dart     # Login form with demo buttons
    home_screen.dart      # Bottom nav + screen routing
    dashboard_screen.dart # Stats overview
    attendance_screen.dart # Attendance with date picker
    homework_screen.dart  # Homework list
    marks_screen.dart     # Assessment list
    messaging_screen.dart # Chat conversations + messages
    notifications_screen.dart # Notification list
```

## Note

This project needs Flutter's platform files (android/, ios/, etc.) generated by `flutter create`.
After installing Flutter SDK, run:

```bash
# Generate platform folders in existing project
flutter create --project-name agora_mobile .
```

This will add android/, ios/, web/, etc. without overwriting existing lib/ files.
